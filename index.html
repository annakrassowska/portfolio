O mnie

Cześć! Nazywam się Anna Krassowska. Od dwóch lat aktywnie rozwijam się w dziedzinie Front-end development. Obecnie pracuję w firmie Profitroom na stanowisku Web Support. W pracy mam okazję wykazać się zdyscyplinowaniem, terminowością i kreatywnością.

Poza budową stron internetowych interesuje się grafiką i fotografią. Biegle obsługuje programy z rodziny Adobe - Photoshop, Adobe Xd, Illustrator.

Umiejętności i technologie

HTML

Dobrze znam html, potrafię tworzyć przejrzysty kod i responsywne strony

CSS, SCSS

Sprawnie tworzę style i potrafię używać preprocesora SCSS

Javascript

Znajomość na poziomie podstawowym.

Jquery

Znam i potrafię używać przydatne biblioteki takie jak Aos, slickslide.

Grafika

Biegle obsługuje programy graficzne takie jak Photoshop lub Illustrator.

Wordpress

Potrafię utworzyć stronę na szablonie i builderze Elementor, znam podstawowe wtyczki.

GIT

Znam podstawowe komendy oraz korzystam z pracy na branche'ach

NPM

Korzystam z paczek npm w celu instalacji bibliotek, kompilacji kodu scss oraz jego minifikacji.
